Installation d'OCS Inventory NG

  1. Installez les dépendances nécessaires :
    2. sudo apt install apache2 mariadb-server php libapache2-mod-perl2 ...
  2. Téléchargez et installez OCS :
    3. wget https://github.com/OCSInventory-NG/...
tar xvzf OCSNG_UNIX_SERVER-2.10.0.tar.gz
sudo ./setup.sh

Installation de GLPI

  1. Créez une base de données :
    2. CREATE DATABASE dbglpi;
GRANT ALL PRIVILEGES ON dbglpi.* TO 'glpiuser'@'localhost' IDENTIFIED BY 'password';
  2. Installez GLPI :
    3. wget https://github.com/glpi-project/glpi/releases/download/...
tar xvzf glpi-10.0.3.tgz -C /var/www/html
sudo chown -R www-data:www-data /var/www/html/glpi
